# Floppy + Piezo MIDI Player

3 Floppy drives and 10 Piezo buzzers play MIDI music using a Raspberry PI.

![picture of the setup](kep.jpg)

Demo video: [Star Wars Medley](https://youtu.be/kjZNSoIBvME).
